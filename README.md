## Installation

1. Clone the repository:
    ```sh
    git clone  https://github.com/somilsdiaz/DrivUp
    cd uUnibus   ```

2. Install dependencies:
    ```sh
    npm install
    ```

3. Start the development server:
    ```sh
    npm run dev
    ```
